## 项目信息
本项目是一个基于PHP的轻量的文件下载站.

它是为了 PTMU Servers 而开发的.

**注意:由于 PTMU Servers 已经停止服务，本项目也将会停止支持并归档**

## 使用方法

1. 下载项目文件
2. 编辑 config/config_global.php 和 config/config_data.php
3. 完成！
